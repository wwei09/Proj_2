Screenplay

Act 2 Scene 3 The Job Interview

Film context: In this scene our hero shows her/his poise and subject knowledge of front-end web design and gets the job

 

Question 1:
Interviewer: So, I see from your college transcript that you studied web design...

Yes, through the Web Design course, I learned how to created my own webpages and  basics of HTML, CSS, and JavaScript.

Question 2:
Interviewer: Tell me about the roles of HTML, CSS and javascript

The roles of HTML, CSS, and JavaScript are distinct, but very interwoven. The HTML lays foundations of web site and all of its conent for the web page. The CSS adds styles to these contents that makes each web page unique in its design. The JavaScript adds functions and features to the content that allows the web page to perform multiple tasks.

Question 3:
Interviewer: Was it largely a theory class or was there much hands-on?

The class contained both theory and hands-on, but I would say there were more hands-on. We spend most our class time hams-on creating our own webpages.

Question 4:
Interviewer: What did you enjoy the most?

The most interesting to me would be actually creating my own webpage. Even everything on it is about class assignments, but I actually enjoy the processes of creating the webpages.

Question 5:
Interviewer: Is there anything you would like to add or change?

I think everything turn out great for me, so there's nothing I want to add or change.
